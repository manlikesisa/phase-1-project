In this website a user can generate a meme and also add a meme as long as the user writes meme's title and the author of the meme.Failure to do that the following alert message will be displayed "Please upload an image and provide a title and author".Everttime a meme appears there is also a comment on top of the meme and below the meme there is the author of the meme.The form on the far right allows the user to add meme of their choice and also add the title of the meme and also the author of the meme.In this website you can listen to music while watching and adding memes.You can pause and play the music.
The following API was used as the baseurl("https://meme-api.com/gimme/wholesomememes").
The following link is the deployed link("https://manlikesisa.github.io/phase-1-project/")

The folder's structure:
      -Index.HTML
      -Css(style.css)
      -Index.js
      -README.md

 How each was used:
  -Html was used as the basic website structure in adding the headers,buttons and the sidebar.
  -css was used in styling the whole page by making it appealing to the user.
  -index.js was used in fetching the API for the memes.
  -readme was used i displaying written information about the website.
   

   HTTP operators used:
   -GET
   -POST
   -PUT

   Events used:
   -mouseover
   -Focus
   -Keydown
