In this website a user can generate a meme and also add a meme as long as the user writes meme's title and the author of the meme.Failure to do that the following alert message will be displayed "Please upload an image and provide a title and author".
The following API was used as the baseurl("https://meme-api.com/gimme/wholesomememes").
The following link is the deployed link("https://manlikesisa.github.io/phase-1-project/")

The folder's structure:
      -Index.HTML
      -Css(style.css)
      -Index.js
      -README.md

 How each was used:
  -Html was used as the basic website structure in adding the headers,buttons and the sidebar.
  -css was used in styling the whole page by making it appealing to the user.
  -index.js was used in fetching the API for the memes.
  -readme was used i displaying written information about the website.
   

   HTTP operators used:
   -GET
   -POST
   -PUT

   Events used:
   -mouseover
   -Focus
   -Keydown

   Images in this project were from UNSPLASH.COM
